Project Report

Sierra Luxe - African Fashion E-Commerce Platform

Project Overview

Sierra Luxe is a modern e-commerce platform designed to showcase and sell authentic African fashion. The platform provides a seamless shopping experience for customers while offering robust management tools for administrators.

This project was developed as part of the ITC4214 Internet Programming course, demonstrating full-stack web development skills using modern technologies and best practices.

Technology Stack

Django

What it is: Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design.

Why we chose it:

  • Built-in admin panel for easy content management
  • Robust security features (CSRF protection, SQL injection prevention)
  • ORM (Object-Relational Mapping) for database operations
  • Large community and extensive documentation
  • Perfect for e-commerce applications with complex data models

Vercel Hosting

What it is: Vercel is a cloud platform for static sites and serverless functions that enables developers to host web projects with ease.

Why we chose it:

  • Automatic deployments from Git repositories
  • Global CDN for fast content delivery worldwide
  • Free SSL certificates for secure connections
  • Excellent performance and uptime
  • Seamless integration with Django through serverless functions
  • Easy scaling as the application grows

Supabase

What it is: Supabase is an open-source Firebase alternative that provides a PostgreSQL database, authentication, and real-time subscriptions.

Why we chose it:

  • PostgreSQL database with full SQL capabilities
  • Built-in authentication system (email/password, OAuth)
  • Real-time data synchronization
  • Row-level security for data protection
  • Free tier with generous limits for development
  • Easy integration with Django applications

Cloudinary

What it is: Cloudinary is a cloud-based image and video management service that provides storage, optimization, and delivery of media assets.

Why we chose it:

  • Automatic image optimization and compression
  • On-the-fly image transformations (resize, crop, format)
  • CDN delivery for fast image loading
  • Secure image storage with access controls
  • Generous free tier for small to medium projects
  • Easy Django integration for seamless media management

System Architecture

The application follows a Model-View-Controller (MVC) pattern implemented through Django's MVT (Model-View-Template) architecture:

  • Models: Define the database structure (Products, Categories, Users, Orders, Reviews)
  • Views: Handle business logic and user requests
  • Templates: Render HTML pages with dynamic content
  • URLs: Route incoming requests to appropriate views

The frontend uses Bootstrap 5 for responsive design, jQuery for DOM manipulation, and custom JavaScript for interactive features like carousels and modals.

Conclusion

Sierra Luxe demonstrates a complete e-commerce solution built with modern web technologies. The combination of Django's robust backend, Vercel's scalable hosting, Supabase's database and authentication, and Cloudinary's media management creates a powerful and maintainable platform.

The project showcases skills in full-stack development, database design, API integration, and responsive web design, meeting the requirements of the ITC4214 Internet Programming course.
